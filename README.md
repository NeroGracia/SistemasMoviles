# Ajolotolli

Proyecto final (PIA) de la materia Programación de Sistemas Móviles.

## Descripción
Ajolotolli es una aplicación móvil tipo red social enfocada en la comunicación breve, dinámica y comunitaria. 
Permitirá a los usuarios publicar mensajes cortos, compartir contenido multimedia y comunicarse mediante chats individuales o grupales.

## Tecnologías
- Android Studio  
- Kotlin
